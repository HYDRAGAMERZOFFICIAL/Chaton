================================================================================
TROUBLESHOOTING
================================================================================

"run.bat doesn't work?"
   - Ensure Python 3.9+ is installed
   - Run: python --version
   - Check PATH environment variable

"Port 8000 already in use?"
   - Close other applications using the port
   - Or change port in backend/main.py

"Models not found?"
   - Run: python train_model.py
   - This trains and saves the ML model

"Can't import modules?"
   - Reinstall: pip install -r requirements.txt
   - Check: python test_imports.py

For more help:
   - See STARTUP.md for detailed guide
   - See README.md for complete documentation
   - See COMPLETION_REPORT.md for project details

================================================================================
API EXAMPLES
================================================================================

Send a chat message:
curl -X POST http://localhost:8000/chat/ask \
  -H "Content-Type: application/json" \
  -d "{\"message\": \"Tell me about the college\"}"

Expected response:
{
  "user_query": "Tell me about the college",
  "intent": "about",
  "confidence": 0.85,
  "response": "Sri Satya Sai Group of Colleges...",
  "source": "knowledge_base"
}

View API documentation:
   http://localhost:8000/docs

Check API status:
   http://localhost:8000/health/status
