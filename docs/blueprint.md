# **App Name**: CollegeConnect Chat

## Core Features:

- FAQ Retrieval: Retrieves relevant answers from a predefined JSON FAQ knowledge base based on user queries.
- Similarity Matching: Uses NLP techniques (TF-IDF and Cosine Similarity) to match user questions with the most appropriate questions in the FAQ knowledge base.
- Instant Responses: Provides instant responses to user queries through a web interface built with Streamlit.
- Web Interface: Displays a user-friendly web interface where users can input questions and view answers.
- AI-Powered FAQ Suggestion: Suggests potential follow-up questions or related FAQs to the user using a generative AI tool, enhancing the conversational experience.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, in line with the educational context.
- Background color: Light gray (#F5F5F5) for a clean and neutral backdrop, ensuring readability and focus on the content.
- Accent color: Soft green (#4CAF50) to highlight important information and CTAs, creating a sense of approachability.
- Body font: 'PT Sans', a humanist sans-serif, will be used for the primary text. Headline font: 'Space Grotesk', a computerized sans-serif, will be used for headers to give a balance between approachability and modernity.
- Use simple, clean icons related to college life and academics to enhance visual appeal and guide users.
- Design a clear and intuitive layout with a prominent search bar and easily accessible FAQ sections.
- Incorporate subtle animations to provide feedback on user interactions and guide the user's attention, enhancing the overall user experience.
