EXAMPLE API USAGE
================================================================================

SEND A MESSAGE (POST REQUEST):

Using curl:
  
  curl -X POST http://localhost:8000/chat/ask ^
    -H "Content-Type: application/json" ^
    -d "{\"message\": \"Tell me about the college\"}"

Using Python:

  import requests
  
  response = requests.post(
    'http://localhost:8000/chat/ask',
    json={'message': 'Tell me about the college'}
  )
  
  print(response.json())

RESPONSE WILL BE:

  {
    "user_query": "Tell me about the college",
    "intent": "about",
    "confidence": 0.85,
    "response": "Sri Satya Sai Group of Colleges was founded...",
    "source": "knowledge_base"
  }

SEE API DOCS:

  Open: http://localhost:8000/docs
  Interactive API documentation with Swagger UI

WHAT HAPPENS WHEN YOU RUN run.bat
================================================================================

STEP 1: Check Python
  "Is Python installed?"
  -> Error if no Python found

  "Install required packages..."
  -> Runs: pip install -r requirements.txt

STEP 3: Train Model
  "Training ML model..."
  -> Runs: python train_model.py
  -> Creates model files in backend/models/

STEP 4: Start Backend
  "Starting Backend API..."
  -> Opens new window
  -> Runs: cd backend && python main.py
  -> Backend starts on http://localhost:8000

STEP 5: Start Frontend
  "Starting Frontend UI..."
  -> Opens new window
  -> Runs: cd frontend && streamlit run app.py
  -> Frontend starts on http://localhost:8501
  -> Browser opens automatically

  -> Use the API
